# 恒一听写系统 - 需求文档

## 产品定位
给家长用的智能听写排程器，艾宾浩斯遗忘曲线管复习节奏，家长管选词和报词。

## 用户
- 主要用户：家长（爸爸、妈妈、其他家人）
- 间接使用者：恒一（9岁，三年级下册）

## 核心功能
1. **词库管理**：语文按课（词语优先、生字补充），英语按单元（单词+中文意思）
2. **听写清单**：家长选课/单元 → 勾选词 → 附带遗忘曲线到期复习词
3. **批改反馈**：飞书群拍照 @灵爪 → 识别对错 → 更新仓库
4. **进度总览**：掌握率、高频错词、各课完成情况

## 技术约束
- 纯前端 HTML/CSS/JS，部署到 GitHub Pages
- 数据存仓库 JSON 文件
- 通过飞书群 + 灵爪 AI 更新数据
