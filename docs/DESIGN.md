# 恒一听写系统 - 设计文档

## 仓库结构
```
hengyi-dictation/
├── index.html
├── css/style.css
├── js/app.js
└── data/
    ├── chinese/
    │   ├── lessons.json
    │   └── L01.json
    └── english/
        ├── units.json
        └── U01.json
```

## 页面设计（3页）
1. **听写模式**：选科目 → 选课/单元 → 勾选词 → 生成清单
2. **词库管理**：查看各课/单元词语和进度
3. **进度总览**：掌握率统计、高频错词

## 数据结构
- 轮次：0(新) → 1(1天) → 2(3天) → 3(7天) → 4(15天) → 5(30天) → ✅(掌握)
- 错：回轮次1，对：轮次+1

## 数据流
仓库 JSON → GitHub Pages 读取 → 网页展示 → 灵爪更新 → 仓库提交
