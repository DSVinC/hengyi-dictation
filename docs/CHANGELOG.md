## v0.8.13 - 2026-04-28

### 清债模式 + 本地调试隔离 + 固定英语发音

**新增**：
- 当天 `新词 + 到期复习` 超过 `30` 个时，选词页显示 `🧹 清债模式`
- 普通模式下：按 `R1 / R2+` 占比分配当天复习名额，并在各自分组内优先保留 `wrongCount` 更高的词
- 本地调试环境自动禁用 GitHub 远端同步，右上角显示 `🧪 本地调试`
- 英语单词优先播放固定音频资源，缺失时再回退浏览器系统发音
- 新增英语固定音频批量生成脚本：`scripts/generate-english-audio.mjs`

**交互优化**：
- 点击“生成听写清单”后，页面自动滚动到听写清单区域
- 点击“📝 听写完毕”后，页面自动滚动到“错字词勾选完毕”操作区
- `R2+` 到期复习词改为只读勾选态，避免出现“可勾选但不生效”的误导

**资源与配置**：
- 新增 `data/audio/english/` 固定英语音频资源及 `manifest.json`
- `.gitignore` 增加本地调试截图和浏览器日志目录排除项

**变更文件**：`js/app.js` + `css/style.css` + `index.html` + `scripts/generate-english-audio.mjs` + `data/audio/english/*` + `.gitignore`
